CSS

  • Enter your CSS notes here
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Colors are input with either a basic color name or their hex code (six alphanumeric digits.)
  • It might take a little longer to remember all the ins and outs and CSS, but it will be nice to have that knowledge!

Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: fetches and downloads content from a remote repository and immedaitely updates the local repository to match that content
  • When I told my friends who are programmers that I was starting with Git, they were very pleased to hear that.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Use camelCase to help define variables and function names in JavaScript.
